
# GenLayer Announces First Ever Dual Inference & Validator Partnership with LibertAI and Aleph Cloud

**San Francisco, August 26, 2025** – GenLayer, the Trust Infrastructure for the AI Age, today announced a partnership with LibertAI and Aleph Cloud. For the first time, GenLayer is onboarding an inference provider and validator simultaneously, setting a new precedent in decentralized and private AI inference on the blockchain.

Through this collaboration, LibertAI’s confidential, verifiable inference layer will power GenLayer validators, while Aleph Cloud joins the network as an official validator, securing the very chain that consumes its compute.

“By bringing LibertAI and Aleph Cloud together, we’re proving what trust infrastructure for the AI age really looks like: confidential inference, verifiable validation, and unstoppable decentralization. This partnership sets a new standard for how AI and blockchain can scale together, and it’s only the beginning of what developers will be able to build on GenLayer,” said Albert Castellana, Co-Founder and CEO of GenLayer Labs.  

## **Unlocking Privacy-First Intelligent Contracts**

The collaboration between GenLayer, LibertAI, and Aleph Cloud introduces a new class of privacy-first Intelligent Contracts powered by AI inference inside Trusted Execution Environments (TEEs).

Unlike traditional inference, where prompts and outputs can be exposed, LibertAI ensures that every validator prompt and output remains private. Running inside AMD SEV-SNP secured TEEs, inference requests generate cryptographic attestations that are stored with the transaction itself. This allows validators to prove that an inference was executed correctly  without revealing sensitive data 

**This privacy guarantee is critical for use cases in:**

- Finance: Confidential AI-driven arbitration for disputes over sensitive transactions, loans, or derivative contracts.  
    
- Prediction Markets: Private queries where validators resolve outcomes without exposing market-moving information.  
    
- Parametric Insurance: AI inference on weather, event, or catastrophe data without leaking customer inputs or claims.  
    
- DAOs & Governance: Private proposals and votes resolved through AI inference while protecting community member identities.  
    

Meanwhile, Aleph Cloud’s validator role demonstrates the robustness of its infrastructure: it not only provides compute for LibertAI, but also secures the very chain consuming that compute. This tight feedback loop of inference and validation ensures that GenLayer’s arbitration remains both confidential and production-grade, offering developers and organizations a scalable foundation for deploying AI-native applications.

Together, the three projects create a privacy-first, scalable AI ecosystem that strengthens trust, security, and reliability across decentralized systems. This sets a new benchmark for how inference and validation should operate in tandem.

## Why It Matters

Traditional arbitration in decentralized systems often relies on centralized LLM providers or human intermediaries. These approaches expose sensitive data, introduce risks of bias, and create costly bottlenecks. By contrast, LibertAI’s no-KYC, open-source platform leverages Aleph Cloud’s decentralized GPU network and TEEs to deliver private, verifiable inference at scale. This combination eliminates single points of failure while enabling developers worldwide to build without centralized oversight.

The GenLayer–LibertAI–Aleph Cloud partnership addresses these challenges head-on:

- **Confidential Inference:** LibertAI’s LLMs run inside AMD SEV-SNP secured TEEs, ensuring prompts and outputs remain private to the caller.  
    
- **Validator Integration:** Aleph Cloud powers inference and secures the GenLayer chain as a validator, proving the robustness of its infrastructure.  
    
- **AI Credits for Validators:** At launch, every GenLayer testnet validator will receive free inference credits via an allow-list form, making it simple to experiment with privacy-first inference.  
    
- **Scalable Arbitration:** With both inference and validation unified, GenLayer can deliver unbiased, rapid dispute resolution at scale.  
    

This dual integration marks a turning point in AI-native trust infrastructure, which enables Intelligent Contracts to execute complex, multi-model workflows without compromising security or decentralization.

## Next Steps

The integration will debut on Testnet Asimov, with expanded testing planned for Testnet Bradbury and full mainnet alignment in later stages. Developers and node operators will be able to benchmark models, deploy Intelligent Contracts, and access remote-attested inference in real-world use cases.

### About LibertAI

[LibertAI](https://libertai.io/) is a decentralized AI platform designed to be more secure, accessible, resilient, and efficient than traditional centralized alternatives, while reducing bias and protecting user privacy. Its open source large language models and confidential AI platform run on a stack of technologies including IPFS and TEEs (Trusted Execution Environment) in [Aleph Cloud](https://aleph.cloud/), a cross-chain decentralized infrastructure for storage, compute, and AI. This foundation enables LibertAI to operate on a fully decentralized, uncensorable, secure, and resilient network that is practically unstoppable.  
Learn more at [libertai.io](https://libertai.io/).

### About Aleph Cloud

Aleph Cloud offers cross-chain decentralized infrastructure for storage, compute, and AI. Built on hundreds of globally distributed nodes, it powers scalable and resilient applications for Web3, DeFi, AI, and gaming industries. Developers rely on Aleph Cloud’s marketplace for high-performance resources across networks like Ethereum, Solana, Base, and Avalanche. By running a GenLayer validator, Aleph Cloud proves its production-grade compatibility, securing the very chain that consumes its compute power while also enabling LibertAI’s privacy-first AI solutions.  
Learn more at [aleph.cloud](https://aleph.cloud/).
