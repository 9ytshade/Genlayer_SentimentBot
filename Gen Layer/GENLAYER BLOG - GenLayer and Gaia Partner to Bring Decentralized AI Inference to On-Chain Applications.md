# GenLayer and Gaia Partner to Bring Decentralized AI Inference to On-Chain Applications

_Integration unlocks scalable, verifiable AI execution at the edge, enabling a new class of autonomous agents and trustless applications._

**San Francisco – June 18, 2025** – GenLayer, the trust infrastructure for AI, today announced a strategic partnership with Gaia, the decentralized AI inference network built for the edge. This marks the first integration of decentralized edge inference with on-chain arbitration and consensus, offering developers a powerful, scalable, and verifiable execution layer for AI-native applications.

The collaboration enables GenLayer validators to offload compute-intensive AI inference tasks to Gaia nodes. These nodes return model outputs, which are then verified and validated through GenLayer’s multi-model Optimistic Democracy consensus mechanism. The result is a censorship-resistant, cryptographically secure inference pipeline that runs in real time.

“With Gaia, we’re not just scaling inference, we’re decentralizing intelligence,” said David Ruidor, CEO of the GenLayer Foundation. “Their edge-native, OpenAI-compatible infrastructure is a perfect match for our Intelligent Contracts. This integration lets agents act, reason, and be held accountable entirely on-chain.”

By integrating with GenLayer, Gaia becomes a first-class inference backend for on-chain AI systems. GenLayer developers can now route live contract calls to Gaia nodes, bringing adoption, visibility, and network usage to GaiaNet’s infrastructure.

“Every great partnership is only as strong as the people behind it. The GenLayer team sets a high bar - visionary, principled, and relentless. I’m proud to build this future together as we bring trustless, decentralized AI to on-chain applications and enable agents that can reason, resolve, and earn for everyone,” said Matt Wright, Co-Founder of Gaia.

### **Key Integration Highlights:**

- **Decentralized AI Oracles**: GenLayer’s Intelligent Contracts can now query Gaia nodes as decentralized inference oracles, fetching outputs from fine-tuned or domain-specific models.
- **Edge-Based Inference**: Gaia nodes run locally on WasmEdge-compatible devices, preserving data privacy, minimizing latency, and resisting centralized control.
- **On-Chain Verifiability**: Inference results from Gaia agents are verified through GenLayer’s multi-LLM validator consensus system—providing cryptographic assurance without relying on a single model or provider.
- **Real-Time Agent Logic**: Gaia’s support for persistent memory and retrieval-augmented generation (RAG) enables long-form reasoning for on-chain agents, unlocking new use cases like AI-driven DAOs, parametric insurance, and legal agents.

As AI agents increasingly operate across financial, governance, and operational domains, the need for verifiable execution and trusted inference is critical. This partnership delivers both, at scale, on-chain, and at the edge.
